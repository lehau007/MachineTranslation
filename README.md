## 🚀 Getting Started

- Clone the repository:
  ```bash
  git clone https://github.com/lehau007/MachineTranslation.git
  cd MachineTranslation
  ```

- Set up your environment (choose venv/conda/poetry).

- Install project dependencies: transformers, torch
